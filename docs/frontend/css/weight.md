# CSS选择器权重


## 权重值

1. 最高：`!important`

2. 第一：内联样式，权重值：`1000`

3. 第二：ID选择器，权重值：`0100`

4. 第三：类、伪类、属性选择器，权重值：`0010`

5. 第四：元素、伪元素选择器，权重值：`0001`

6. 第五：通配符、子选择器、相邻选择器、兄弟选择器、继承样式，权重值：`0000`

## 权重计算

- `CSS` 自上而下执行，同权重的情况下后面的样式覆盖前面的样式

- 带有上下文关系的选择器比单纯的元素选择器权重要高
